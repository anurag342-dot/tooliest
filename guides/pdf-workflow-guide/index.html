PDF Workflow Guide: Merge, Split, Compress, and Protect PDFs in the Browser

By Anurag April 29, 2026 ~8 min read

PDFs are the universal document format — contracts, invoices, reports, and applications all rely on them. But working with PDFs has traditionally required expensive desktop software or uploading sensitive files to random web services. Neither is ideal when you're dealing with contracts, financial documents, or personal records.

I built Tooliest's suite of 15 PDF tools specifically to solve this problem. Every tool runs entirely in your browser using PDF.js (Mozilla's open-source PDF renderer) and pdf-lib. Your files never leave your device — not even temporarily. Here's how to handle every common PDF task.

1. Merging Multiple PDFs into One

When to use: Combining separate documents into a single file for submission, archiving, or sharing. Common scenarios include merging cover letter + resume, combining invoices for a quarterly report, or assembling a multi-section proposal.

How it works: The merger reads each PDF's internal page tree, copies all pages into a new document in the order you specify, and preserves the original formatting, fonts, and embedded images. Bookmarks and form fields from the source documents are maintained where possible.

Tips for better results:

  • Arrange files in the desired order before merging — it's faster than reordering pages after the fact
  • If one document is landscape and another is portrait, the merged PDF preserves each page's individual orientation
  • For very large merges (50+ pages), the browser may use significant memory. Close other tabs if you notice slowdowns
🔧 Merge PDFs: Tooliest PDF Merger — Drag and drop multiple files, reorder them, and download a single combined PDF.

2. Splitting a PDF into Separate Files

When to use: Extracting specific pages from a large document. Examples: pulling a single chapter from an ebook, separating individual invoices from a batch export, or isolating signature pages for separate distribution.

How it works: You specify which pages to extract (e.g., "1-3, 7, 12-15"), and the tool creates a new PDF containing only those pages. The original file is never modified.

🔧 Split PDFs: Tooliest PDF Splitter — Enter page ranges and extract them instantly.

3. Compressing PDFs for Email and Upload

When to use: When a PDF exceeds email attachment limits (typically 25MB for Gmail), upload restrictions on government or job application portals, or when storage space matters.

How compression works: PDF compression primarily targets embedded images, which are usually the largest component of a PDF's file size. The compressor re-encodes images at a lower quality setting and optionally removes unused fonts, metadata, and redundant objects from the PDF's internal structure.

4. Rotating and Reordering Pages

When to use: Fixing scanned documents where some pages are sideways or upside down, or rearranging pages that were scanned out of order.

Rotation works on individual pages or all pages at once. You can rotate 90°, 180°, or 270° clockwise. Reordering lets you drag pages into the correct sequence before saving.

5. Adding Watermarks

When to use: Marking documents as "DRAFT", "CONFIDENTIAL", or "SAMPLE" before distribution. Watermarks discourage unauthorized redistribution and make it clear that a document isn't final.

Options: Text watermarks (custom text, font size, color, opacity, rotation angle) or image watermarks (company logo overlay). Both are applied non-destructively — the original content underneath remains intact.

🔧 Add watermarks: Tooliest PDF Watermark — Customize text, position, opacity, and color.

6. Password-Protecting PDFs

When to use: Securing sensitive documents like financial statements, medical records, or legal agreements before sending them via email or shared drives.

PDF encryption uses AES-256, which is the same encryption standard used by banks and government agencies. You set a password, and recipients must enter it to open the file. Without the password, the PDF's content is cryptographically inaccessible — not just hidden behind a dialog box.

Important: If the recipient forgets the password, neither you nor Tooliest can recover it. The encryption happens entirely in your browser, so no password is ever transmitted or stored anywhere. Always share the password through a separate channel (e.g., send the PDF via email and the password via text message).

🔧 Protect PDFs: Tooliest PDF Password Protection — AES-256 encryption, entirely in your browser.

7. Converting Between PDFs and Images

Sometimes you need to go between formats:

  • Images → PDF: When you need to combine photos, screenshots, or scans into a single document for submission. Try Images to PDF →
  • PDF → Text: When you need to extract the text content from a PDF for editing, translation, or analysis. Try PDF to Text →
  • Text → PDF: When you need to create a clean PDF from plain text. Try Text to PDF →

Privacy: Why Browser-Based Matters for PDFs

Most online PDF tools upload your file to a server, process it remotely, and return the result. Your sensitive documents pass through infrastructure you don't control, potentially stored in logs, cached by CDNs, or accessible to employees of the service.

Tooliest's PDF tools run entirely in your browser using JavaScript. The files you select are read by your browser's local File API, processed in memory, and the result is generated as a local download. At no point does any file data leave your device. You can verify this by opening your browser's Network tab while using any tool — you'll see zero file upload requests.
